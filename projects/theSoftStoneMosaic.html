The Soft Stone Mosaic

See the GitHub repo

This is one of the projects I'm currently working on. I'm designing the website for a new artistic mosaics business: this is the stage for the artist to showcase and promote her work.

I built the mock-up using Adobe XD. In the design I focused on the User Experience: each section lays out the content clearly, and the user always has a clear idea of where they are in the site. From the home page, the user enters directly inside the Gallery, where all works are displayed with a list of photographs. From here, the user can learn more about each work by going to the individual work's page, where they can read a detailed description and see larger pictures in the carousel.

The Materials page lets the user peek behind the scenes and discover what goes into making a beautiful artistic mosaic. Spoiler alert: it's about quality materials and really hard work!

Finally, the Contacts page is always one click away, so the user can email the artist directly and request further information.


Tool: Adobe XD.
